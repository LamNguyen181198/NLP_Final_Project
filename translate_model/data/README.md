# Dataset format

Use JSONL files with these fields:

- `source`: Vietnamese input text
- `target`: English translation

Example:

```jsonl
{"source": "Xin chào, bạn khỏe không?", "target": "Hello, how are you?"}
{"source": "Tôi đang học tiếng Anh.", "target": "I am learning English."}
```

You can start with a small curated dataset and expand it later.
